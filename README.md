### HOW TO RUN APP
Recommend Node version: `v12.x`
1. Clone repo: 

```
git clone git@github.com:ningo-agilityio/rxjs.git
```

2. Install dependencies

```
npm install
```

3. Start server

```
npm start
```

4. Access port localhost:8080

### HOW TO RUN SERVER
1. Change to server folder
```
cd server
```

2. Install dependencies

```
npm install
```

3. Start server

```
npm start
```

4. Server will start at port localhost:3000
